# Memory

Durable lessons for this Codex workflow. Add entries sparingly and keep them useful across projects.

[LEARN:workflow] Keep plans, checkpoints, and session logs in `quality_reports/` so long-running academic work can resume without relying on chat history.

[LEARN:verification] For slide and manuscript work, always pair edits with the closest mechanical check: LaTeX compile, Quarto render, palette sync, quality score, bibliography validation, or reproducibility audit.

[LEARN:scope] Keep template-level guidance general. Project-specific preferences belong in the fork's `AGENTS.md`, `.codex/rules/knowledge-base-template.md`, or local notes that are not shared back upstream.
